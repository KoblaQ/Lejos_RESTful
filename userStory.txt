 user stories to the web UI

As a user I can set speed
    Through a web browser, a user can set speed to the Lego robot (or slow down or speed up the velocity). The setting is saved into a database. 

As a user I can set at what light intensity the robot will detect the line
    Through a web browser, a user can set the value for light intensity that the robot will use to stay on track.


As a user I can set how many times the robot will follow the line before it celebrates
    Through a web browser, a user can set the count for the cycle which initiates the robot's celebration

As a user I can set, which song will play once it starts celebrating
    Through a web browser, a user can choose which song from closed options that could play once the robot finishes it's cycle around the track.

As a user I can set, what message will be displayed on the screen once it is done with the celebration
    Through a web browser, a user can choose the message that will appear on the screen at the end of the robot's celebration.


user stories to the lego robot

Lego robot can read data from a RESTful web service
    Lego robot has a thread, which reads data, e.g. new speed setting, from a RESTful web service
